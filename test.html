Backend Project

This site was created for our group's backend project. The concept revolves around a blog for pepper enthusiasts. If you have a passion for peppers, you can explore more information about them, discover new pepper varieties, and engage in discussions while also rating peppers on the blog. I was responsible for contributing most of the HTML and handling all of the CSS and styling for the page.

Capstone Project

This website was created as our final Capstone project, bringing together all the skills and knowledge I had acquired throughout the Digital Crafts bootcamp. It's a virtual pet game inspired by the classic Tamagotchi. I was responsible for designing and styling the entire page, using tools like Blender and Procreate to create all the visuals. The project was built using React, CSS, and ElephantSQL.
